# word-config

Thư mục này chứa template cho xuất file Word.

## reference.docx

Đặt file `reference.docx` vào đây để định nghĩa style cho file Word xuất ra.

### Cách tạo reference.docx:
1. Tạo file Word mới
2. Thiết lập: A4, Lề (Trái 1.5cm, Phải 1cm, Trên 1.3cm, Dưới 1.3cm)
3. Font mặc định: Times New Roman 12pt, Line spacing 1.3
4. Lưu thành `reference.docx` và đặt vào thư mục này

Nếu không có file này, pandoc vẫn tạo được .docx với style mặc định.
